Dashboard йЎөйқўд»Ӣз»Қ

Dashboard йЎөйқўдё»иҰҒеҲҶдёәжңәеҷЁгҖҒжңҚеҠЎгҖҒз®ЎзҗҶдёүдёӘйғЁеҲҶпјҢжң¬ж–Үе°ҶиҜҰз»Ҷд»Ӣз»Қиҝҷдәӣз•ҢйқўгҖӮ

йЎөйқўжҰӮи§ҲвҖӢ

жңәеҷЁйЎөйқўд»Ӣз»ҚвҖӢ

еҚ•еҮ»жңәеҷЁ->жҰӮи§Ҳиҝӣе…ҘжңәеҷЁжҰӮи§ҲйЎөйқўгҖӮ

з”ЁжҲ·еҸҜеҝ«йҖҹжҹҘзңӢ CPUгҖҒMemoryгҖҒLoadгҖҒDisk е’Ң Network In/Out еҸҳеҢ–жғ…еҶөгҖӮ

  • й»ҳи®ӨжңҖеӨҡеҸҜйҖүжӢ© 14 еӨ©зҡ„зӣ‘жҺ§ж•°жҚ®иҝӣиЎҢжҹҘзңӢпјҢж”ҜжҢҒйҖүжӢ©ж—¶й—ҙж®өпјҢд№ҹеҸҜд»Ҙеҝ«жҚ·йҖүжӢ©жңҖиҝ‘ 1 е°Ҹж—¶гҖҒ6 е°Ҹж—¶гҖҒ12 е°Ҹж—¶гҖҒ1 еӨ©гҖҒ3 еӨ©гҖҒ7 еӨ©е’Ң 14 еӨ©гҖӮ

  • й»ҳи®ӨеӢҫйҖүжҳҫзӨәжүҖжңүе®һдҫӢзҡ„зӣ‘жҺ§ж•°жҚ®пјҢеҸҜд»ҘеңЁе®һдҫӢжЎҶеҶ…и°ғж•ҙгҖӮ

  • йЎөйқўзҡ„зӣ‘жҺ§ж•°жҚ®й»ҳи®ӨдёҚиҮӘеҠЁжӣҙж–°пјҢеҸҜд»Ҙи°ғж•ҙжӣҙж–°йў‘зҺҮи®©йЎөйқўиҮӘеҠЁжӣҙж–°пјҢд№ҹеҸҜд»ҘеҚ•еҮ»жҢүй’®жүӢеҠЁжӣҙж–°гҖӮ

  • еҰӮжһңйңҖиҰҒи®ҫзҪ®еҹәзәҝпјҢдҪңдёәеҸӮиҖғж ҮеҮҶзәҝпјҢеҸҜд»ҘеҚ•еҮ»жЁЎеқ—еҸідёҠи§’зҡ„ жҢүй’®гҖӮ

  • еҰӮжһңйңҖиҰҒжҹҘзңӢжҹҗдёҖйЎ№жӣҙиҜҰз»Ҷзҡ„зӣ‘жҺ§жҢҮж ҮпјҢеҸҜд»ҘеҚ•еҮ»жЁЎеқ—еҸідёҠи§’зҡ„ жҢүй’®пјҢеңЁзӨәдҫӢдёӯйҖүжӢ©LoadжҹҘзңӢиҜҰжғ…дҝЎжҒҜпјҢеҰӮдёӢеӣҫгҖӮ

    • еҸҜд»Ҙи®ҫзҪ®зӣ‘жҺ§ж—¶й—ҙж®өгҖҒе®һдҫӢгҖҒжӣҙж–°йў‘зҺҮе’ҢеҹәзәҝгҖӮ
    • еҸҜд»Ҙжҗңзҙўе’ҢеӢҫйҖүжҢҮж ҮгҖӮзӣ‘жҺ§жҢҮж ҮиҜҰжғ…иҜ·еҸӮи§Ғзӣ‘жҺ§жҢҮж ҮиҜҙжҳҺгҖӮ
    • еҸҜд»ҘжҡӮж—¶йҡҗи—ҸдёҚйңҖиҰҒжҹҘзңӢзҡ„иҠӮзӮ№гҖӮ
    • еҸҜд»ҘеҚ•еҮ» жҢүй’®жҹҘзңӢжҢҮж ҮиҜҰжғ…гҖӮ

жңҚеҠЎйЎөйқўд»Ӣз»ҚвҖӢ

еҚ•еҮ»жңҚеҠЎ->жҰӮи§Ҳиҝӣе…ҘжңҚеҠЎжҰӮи§ҲйЎөйқўгҖӮ

з”ЁжҲ·еҸҜеҝ«йҖҹжҹҘзңӢ GraphгҖҒMetaгҖҒStorage жңҚеҠЎзҡ„дҝЎжҒҜгҖӮ

еӨҮжіЁ

жңҚеҠЎзӣ‘жҺ§йЎөд»…ж”ҜжҢҒжҜҸз§ҚжңҚеҠЎи®ҫзҪ®дёӨдёӘзӣ‘жҺ§жҢҮж ҮпјҢеҸҜд»ҘеҚ•еҮ»жЁЎеқ—еҶ…зҡ„и®ҫзҪ®жҢүй’®иҝӣиЎҢи°ғж•ҙгҖӮ

  • й»ҳи®ӨжңҖеӨҡеҸҜйҖүжӢ© 14 еӨ©зҡ„зӣ‘жҺ§ж•°жҚ®иҝӣиЎҢжҹҘзңӢпјҢж”ҜжҢҒйҖүжӢ©ж—¶й—ҙж®өпјҢд№ҹеҸҜд»Ҙеҝ«жҚ·йҖүжӢ©жңҖиҝ‘ 1 е°Ҹж—¶гҖҒ6 е°Ҹж—¶гҖҒ12 е°Ҹж—¶гҖҒ1 еӨ©гҖҒ3 еӨ©гҖҒ7 еӨ©е’Ң 14 еӨ©гҖӮ

  • й»ҳи®ӨеӢҫйҖүжҳҫзӨәжүҖжңүе®һдҫӢзҡ„зӣ‘жҺ§ж•°жҚ®пјҢеҸҜд»ҘеңЁе®һдҫӢжЎҶеҶ…и°ғж•ҙгҖӮ

  • йЎөйқўзҡ„зӣ‘жҺ§ж•°жҚ®й»ҳи®ӨдёҚиҮӘеҠЁжӣҙж–°пјҢеҸҜд»Ҙи°ғж•ҙжӣҙж–°йў‘зҺҮи®©йЎөйқўиҮӘеҠЁжӣҙж–°пјҢд№ҹеҸҜд»ҘеҚ•еҮ»жҢүй’®жүӢеҠЁжӣҙж–°гҖӮ

  • еҸҜд»ҘжҹҘзңӢйӣҶзҫӨеҶ…жүҖжңүжңҚеҠЎзҡ„зҠ¶жҖҒгҖӮ

  • еҰӮжһңйңҖиҰҒжҹҘзңӢжҹҗдёҖйЎ№жӣҙиҜҰз»Ҷзҡ„зӣ‘жҺ§жҢҮж ҮпјҢеҸҜд»ҘеҚ•еҮ»жЁЎеқ—еҸідёҠи§’зҡ„ жҢүй’®пјҢеңЁзӨәдҫӢдёӯйҖүжӢ© Graph жҹҘзңӢиҜҰжғ…дҝЎжҒҜпјҢеҰӮдёӢеӣҫгҖӮ

    • еҸҜд»Ҙи®ҫзҪ®зӣ‘жҺ§ж—¶й—ҙж®өгҖҒе®һдҫӢгҖҒжӣҙж–°йў‘зҺҮгҖҒе‘ЁжңҹгҖҒиҒҡеҗҲж–№ејҸе’ҢеҹәзәҝгҖӮ
    • еҸҜд»Ҙжҗңзҙўе’ҢеӢҫйҖүжҢҮж ҮгҖӮзӣ‘жҺ§жҢҮж ҮиҜҰжғ…иҜ·еҸӮи§Ғзӣ‘жҺ§жҢҮж ҮиҜҙжҳҺгҖӮ
    • еҸҜд»ҘжҡӮж—¶йҡҗи—ҸдёҚйңҖиҰҒжҹҘзңӢзҡ„иҠӮзӮ№гҖӮ
    • еҸҜд»ҘеҚ•еҮ» жҢүй’®жҹҘзңӢжҢҮж ҮиҜҰжғ…гҖӮ
    • Graph жңҚеҠЎж”ҜжҢҒдёҖзі»еҲ—еҹәдәҺеӣҫз©әй—ҙзҡ„зӣ‘жҺ§жҢҮж ҮгҖӮиҜҰжғ…еҸӮи§ҒдёӢж–Үеӣҫз©әй—ҙзӣ‘жҺ§гҖӮ

еӣҫз©әй—ҙзӣ‘жҺ§вҖӢ

иӯҰе‘Ҡ

дҪҝз”Ёеӣҫз©әй—ҙжҢҮж ҮеүҚпјҢз”ЁжҲ·йңҖиҰҒеңЁ Graph жңҚеҠЎдёӯпјҢи®ҫзҪ®enable_space_level_metricsдёәtrueгҖӮе…·дҪ“ж“ҚдҪңпјҢеҸӮи§ҒGraph жңҚеҠЎй…ҚзҪ®гҖӮ

:::compatibility еӣҫзӣ‘жҺ§жҢҮж ҮдёҚе…је®№жҖ§

еҰӮжһңеӣҫз©әй—ҙзҡ„еҗҚз§°еҢ…жӢ¬зү№ж®Ҡеӯ—з¬ҰпјҢеҸҜиғҪдјҡжңүеӣҫз©әй—ҙзӣ‘жҺ§жҢҮж Үж•°жҚ®дёҚжҳҫзӨәзҡ„й—®йўҳгҖӮ

::: жңҚеҠЎзӣ‘жҺ§йЎөйқўиҝҳеҸҜд»Ҙзӣ‘жҺ§еӣҫз©әй—ҙзә§еҲ«жҢҮж ҮгҖӮеҸӘжңүеҪ“еӣҫз©әй—ҙжҢҮж Үзҡ„иЎҢдёәиў«и§ҰеҸ‘еҗҺпјҢз”ЁжҲ·жүҚиғҪжҢҮе®ҡеӣҫз©әй—ҙжҹҘзңӢеҜ№еә”зҡ„еӣҫз©әй—ҙжҢҮж Үзҡ„дҝЎжҒҜгҖӮеӣҫз©әй—ҙзҡ„зӣ‘жҺ§жҢҮж Үи®°еҪ•дёҚеҗҢеӣҫз©әй—ҙзҡ„ж•°жҚ®гҖӮзӣ®еүҚпјҢеҸӘжңү Graph жңҚеҠЎж”ҜжҢҒеҹәдәҺеӣҫз©әй—ҙзҡ„зӣ‘жҺ§жҢҮж ҮгҖӮ

Dashboard ж”ҜжҢҒзҡ„еӣҫз©әй—ҙжҢҮж ҮпјҢеҸӮи§Ғеӣҫз©әй—ҙзә§еҲ«зӣ‘жҺ§жҢҮж ҮгҖӮ

з®ЎзҗҶйЎөйқўд»Ӣз»ҚвҖӢ

дҝЎжҒҜжҖ»и§ҲвҖӢ

з”ЁжҲ·еҸҜд»ҘеңЁдҝЎжҒҜжҖ»и§ҲйЎөйқўжҹҘзңӢ жӮҰж•°еӣҫж•°жҚ®еә“ еҶ…ж ёзӣёе…ідҝЎжҒҜпјҢеҢ…жӢ¬ Storage жңҚеҠЎдҝЎжҒҜгҖҒStorage Leader еҲҶеёғгҖҒ жӮҰж•°еӣҫж•°жҚ®еә“ еҗ„дёӘжңҚеҠЎзҡ„зүҲжң¬дҝЎжҒҜеҸҠжүҖеңЁиҠӮзӮ№дҝЎжҒҜгҖҒеҲҶзүҮзҡ„еҲҶеёғжғ…еҶөеҸҠиҜҰз»ҶдҝЎжҒҜгҖӮ

Storage Leader еҲҶеёғвҖӢ

жҳҫзӨә Leader ж•°йҮҸеҸҠ Leader зҡ„еҲҶеёғгҖӮ

  • еҚ•еҮ»еҸідёҠи§’зҡ„ Balance Leader жҢүй’®еҸҜд»Ҙеҝ«йҖҹеңЁ жӮҰж•°еӣҫж•°жҚ®еә“ йӣҶзҫӨдёӯеқҮиЎЎеҲҶеёғ LeaderгҖӮе…ідәҺ Leader зҡ„иҜҰз»ҶдҝЎжҒҜпјҢеҸӮи§Ғ Storage жңҚеҠЎгҖӮ

  • еҚ•еҮ»еҸідёҠи§’зҡ„иҜҰжғ…пјҢжҹҘзңӢ Leader еҲҶеёғзҡ„иҜҰз»ҶдҝЎжҒҜгҖӮ

зүҲжң¬вҖӢ

жҳҫзӨә жӮҰж•°еӣҫж•°жҚ®еә“ жүҖжңүжңҚеҠЎзүҲжң¬еҸҠжңҚеҠЎең°еқҖгҖӮеҚ•еҮ»еҸідёҠи§’зҡ„иҜҰжғ…пјҢжҹҘзңӢжӣҙеӨҡдҝЎжҒҜгҖӮ
