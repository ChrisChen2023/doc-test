错误码

NebulaGraph 运行出现问题时，会返回错误码。本文介绍错误码的详细信息。

备注

  • 如果出现错误但没有返回错误码，或错误码描述不清，请在论坛或 GitHub 反馈。

  • 返回0表示执行成功。
