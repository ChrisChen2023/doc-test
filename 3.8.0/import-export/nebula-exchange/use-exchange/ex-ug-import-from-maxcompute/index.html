еҜје…Ҙ MaxCompute ж•°жҚ®В¶

жң¬ж–Үд»ҘдёҖдёӘзӨәдҫӢиҜҙжҳҺеҰӮдҪ•дҪҝз”Ё Exchange е°ҶеӯҳеӮЁеңЁ MaxCompute дёҠзҡ„ж•°жҚ®еҜје…Ҙ NebulaGraph гҖӮ

ж•°жҚ®йӣҶВ¶

жң¬ж–Үд»Ҙ basketballplayer ж•°жҚ®йӣҶдёәдҫӢгҖӮ

зҺҜеўғй…ҚзҪ®В¶

жң¬ж–ҮзӨәдҫӢеңЁ MacOS дёӢе®ҢжҲҗпјҢд»ҘдёӢжҳҜзӣёе…ізҡ„зҺҜеўғй…ҚзҪ®дҝЎжҒҜпјҡ

  • зЎ¬д»¶и§„ж јпјҡ
    • CPUпјҡ1.7 GHz Quad-Core Intel Core i7
    • еҶ…еӯҳпјҡ16 GB
  • Sparkпјҡ2.4.7пјҢеҚ•жңәзүҲ
  • MaxComputeпјҡйҳҝйҮҢдә‘е®ҳж–№зүҲжң¬
  • NebulaGraph пјҡ3.8.0гҖӮ

еүҚжҸҗжқЎд»¶В¶

ејҖе§ӢеҜје…Ҙж•°жҚ®д№ӢеүҚпјҢз”ЁжҲ·йңҖиҰҒзЎ®и®Өд»ҘдёӢдҝЎжҒҜпјҡ

  • е·Із»Ҹе®үиЈ…йғЁзҪІ NebulaGraph е№¶иҺ·еҸ–еҰӮдёӢдҝЎжҒҜпјҡ

    • Graph жңҚеҠЎе’Ң Meta жңҚеҠЎзҡ„зҡ„ IP ең°еқҖе’Ңз«ҜеҸЈгҖӮ
    • жӢҘжңү NebulaGraph еҶҷжқғйҷҗзҡ„з”ЁжҲ·еҗҚе’ҢеҜҶз ҒгҖӮ
  • е·Із»Ҹзј–иҜ‘ ExchangeгҖӮиҜҰжғ…иҜ·еҸӮи§Ғзј–иҜ‘ ExchangeгҖӮжң¬зӨәдҫӢдёӯдҪҝз”Ё Exchange 3.8.0гҖӮ
  • е·Із»Ҹе®үиЈ… SparkгҖӮ
  • дәҶи§Ј NebulaGraph дёӯеҲӣе»ә Schema зҡ„дҝЎжҒҜпјҢеҢ…жӢ¬ Tag е’Ң Edge type зҡ„еҗҚз§°гҖҒеұһжҖ§зӯүгҖӮ

ж“ҚдҪңжӯҘйӘӨВ¶

жӯҘйӘӨ 1пјҡеңЁ NebulaGraph дёӯеҲӣе»ә SchemaВ¶

еҲҶжһҗж•°жҚ®пјҢжҢүд»ҘдёӢжӯҘйӘӨеңЁ NebulaGraph дёӯеҲӣе»ә Schemaпјҡ

  1. зЎ®и®Ө Schema иҰҒзҙ гҖӮ NebulaGraph дёӯзҡ„ Schema иҰҒзҙ еҰӮдёӢиЎЁжүҖзӨәгҖӮ

    иҰҒзҙ  еҗҚз§° еұһжҖ§
    Tag player name string, age int
    Tag team name string
    Edge Type follow degree int
    Edge Type serve start_year int, end_year int

  2. еңЁ NebulaGraph дёӯеҲӣе»әдёҖдёӘеӣҫз©әй—ҙ basketballplayerпјҢе№¶еҲӣе»әдёҖдёӘ SchemaпјҢеҰӮдёӢжүҖзӨәгҖӮ

    ## еҲӣе»әеӣҫз©әй—ҙ
nebula> CREATE SPACE basketballplayer \
        (partition_num = 10, \
        replica_factor = 1, \
        vid_type = FIXED_STRING(30));

## йҖүжӢ©еӣҫз©әй—ҙ basketballplayer
nebula> USE basketballplayer;

## еҲӣе»ә Tag player
nebula> CREATE TAG player(name string, age int);

## еҲӣе»ә Tag team
nebula> CREATE TAG team(name string);

## еҲӣе»ә Edge type follow
nebula> CREATE EDGE follow(degree int);

## еҲӣе»ә Edge type serve
nebula> CREATE EDGE serve(start_year int, end_year int);

жӯҘйӘӨ 3пјҡеҗ‘ NebulaGraph еҜје…Ҙж•°жҚ®В¶

иҝҗиЎҢеҰӮдёӢе‘Ҫд»Өе°Ҷ MaxCompute ж•°жҚ®еҜје…ҘеҲ° NebulaGraph дёӯгҖӮе…ідәҺеҸӮж•°зҡ„иҜҙжҳҺпјҢиҜ·еҸӮи§ҒеҜје…Ҙе‘Ҫд»ӨеҸӮж•°гҖӮ

${SPARK_HOME}/bin/spark-submit --master "local" --class com.vesoft.nebula.exchange.Exchange <nebula-exchange.jar_path> -c <maxcompute_application.conf_path>

Note

JAR еҢ…жңүдёӨз§ҚиҺ·еҸ–ж–№ејҸпјҡиҮӘиЎҢзј–иҜ‘жҲ–иҖ…д»Һ maven д»“еә“дёӢиҪҪгҖӮ

зӨәдҫӢпјҡ

${SPARK_HOME}/bin/spark-submit  --master "local" --class com.vesoft.nebula.exchange.Exchange  /root/nebula-exchange/nebula-exchange/target/nebula-exchange_spark_2.4-3.8.0.jar  -c /root/nebula-exchange/nebula-exchange/target/classes/maxcompute_application.conf

з”ЁжҲ·еҸҜд»ҘеңЁиҝ”еӣһдҝЎжҒҜдёӯжҗңзҙўbatchSuccess.<tag_name/edge_name>пјҢзЎ®и®ӨжҲҗеҠҹзҡ„ж•°йҮҸгҖӮдҫӢеҰӮbatchSuccess.follow: 300гҖӮ

жӯҘйӘӨ 4пјҡпјҲеҸҜйҖүпјүйӘҢиҜҒж•°жҚ®В¶

з”ЁжҲ·еҸҜд»ҘеңЁ NebulaGraph е®ўжҲ·з«ҜпјҲдҫӢеҰӮ NebulaGraph Studioпјүдёӯжү§иЎҢжҹҘиҜўиҜӯеҸҘпјҢзЎ®и®Өж•°жҚ®жҳҜеҗҰе·ІеҜје…ҘгҖӮдҫӢеҰӮпјҡ

LOOKUP ON player YIELD id(vertex);

з”ЁжҲ·д№ҹеҸҜд»ҘдҪҝз”Ёе‘Ҫд»Ө SHOW STATS жҹҘзңӢз»ҹи®Ўж•°жҚ®гҖӮ

жӯҘйӘӨ 5пјҡпјҲеҰӮжңүпјүеңЁ NebulaGraph дёӯйҮҚе»әзҙўеј•В¶

еҜје…Ҙж•°жҚ®еҗҺпјҢз”ЁжҲ·еҸҜд»ҘеңЁ NebulaGraph дёӯйҮҚж–°еҲӣе»әе№¶йҮҚе»әзҙўеј•гҖӮиҜҰжғ…иҜ·еҸӮи§Ғзҙўеј•д»Ӣз»ҚгҖӮ
